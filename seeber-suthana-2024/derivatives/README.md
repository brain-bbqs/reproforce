# Seeber-Suthana-2024 (Derivatives)

This directory contains all outputs from the `../code/generate-figures` scripts, which consists of the data and PNG files for each individual figure from the associated manuscript.

The `figure-data/` subdirectory contains JSON files containing the exact data used in each figure rendering.

The `figures/` subdirectory contains the PNG files.

These files are not the true originals used in the manuscript; rather, they are approximate recreations using Python code reading from the NWB-standardized files.
- The original manuscript used MATLAB for both code and data.
